## Performance Optimizations

1. **Memoization**: WebsiteNode uses React.memo to prevent unnecessary re-renders
2. **Lazy Loading**: Thumbnails load only when nodes are in ready state
3. **Efficient Filtering**: Context aggregation uses filter + map pattern
4. **Single API Call**: YouTube metadata fetched once per video
5. **Fallback Strategy**: Graceful degradation if thumbnails fail to load

## Security Considerations

1. **API Key Protection**: YouTube API key stored in environment variables
2. **URL Validation**: Proper URL parsing and validation before API calls
3. **Iframe Sandboxing**: Regular websites use sandboxed iframes
4. **External Links**: All external links open in new tab with noopener/noreferrer
5. **XSS Prevention**: All user inputs are sanitized through React's built-in protection
